Purpose of this index

Beginners often arrive with one urgent question, but the right answer may live in several resource types. A definition answers what a term means. A checklist answers what to verify. A comparison answers which factors differ between options. A safety framework answers what to do before acting. This index routes the question to the correct resource type so readers do not rely on a single thin answer.

Recommended beginner path

Start with the term you do not understand. If the question is about an ID, read the terminology map first. If the question is about whether a community is clear enough, use the scorecard. If the question is about choosing between app ecosystems, use the comparison matrix. If the question is about a specific public guide, read the canonical guide on the owned WordPress site after reviewing the framework.

This sequence prevents a common mistake: treating a direct answer as a complete due-diligence process. For example, knowing what a club ID is does not tell you whether the rules are clear. Knowing that a club belongs to a larger network does not tell you which support path applies. Knowing which app a community uses does not tell you whether the club fits your schedule or experience level.
