Poker App Club ID Checklist

checklist Last updated 2026-07-09.

A checklist for using PokerBros or ClubGG club IDs more carefully.

Before using a club ID

  • Copy the ID exactly from the guide or support contact.
  • Check whether the ID is for a club, referral, or agent/support path.
  • Make sure the destination club name matches what you expected.

After finding the club

  • Read the rules and support instructions.
  • Compare available games, stakes, and schedule.
  • Do not assume the ID alone proves trustworthiness.
